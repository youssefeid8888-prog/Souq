# 🛒 Souq – E-Commerce Website

A full e-commerce web application built with ASP.NET MVC.

## 🔹 Features
- User-friendly product browsing
- Product details page
- Shopping cart
- Order checkout
- Cash on Delivery

## 🔹 Tech Stack
- ASP.NET MVC
- Entity Framework
- SQL Server
- Bootstrap
