# 🛒 Souq – E-Commerce Web Application

Souq is a simple e-commerce web application built using ASP.NET MVC and Entity Framework (Database First).

## 🚀 Features
- Browse products by category
- Product details page
- Shopping cart
- Checkout
- Cash on Delivery

## 🧰 Technologies
- ASP.NET MVC
- Entity Framework
- SQL Server
- Bootstrap

## ▶️ How to Run
- Open the project in Visual Studio
- Update connection string
- Run the project
